# 月の動きシミュレーター

中学受験用の月の動きと満ち欠けを学ぶためのインタラクティブシミュレーター。

## 機能

- **地球の真上から見た図（北極側）** - 内側リング: 太陽光による実際の照らされ方（右半分が常に明るい）、外側リング: 地球から見た月の形
- **地上から見た月の動き** - 東→南→西の弧に沿った月と太陽の位置、時刻に応じた空の色変化
- **月齢スライダー** - 0〜29.5日で月の公転位置を変更
- **時刻スライダー** - 0〜24時で地球の自転（観測者の位置）を変更
- **再生アニメーション** - 速度x1〜x8で時間経過をシミュレーション
- **プリセットボタン** - 新月・三日月・上弦の月・満月・下弦の月
- **情報表示** - 月の出・南中・月の入り時刻

## 技術スタック

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Biome

## セットアップ

```bash
pnpm install
pnpm dev
```

http://localhost:3000 で開きます。
